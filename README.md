# Next.js basic app with docker container
# Overview
OSS version used in this app is below.

| name | version |
| ---- | ---- |
| Node.js | 20.8.0 |
| Next.js | 13.5.4 |

# Getting started
1. Build and start container.
```bash
$ docker-compose build
$ docker-compose up -d
```

2. Open http://localhost:4211/ in browser.
